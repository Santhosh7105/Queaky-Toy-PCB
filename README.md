# Queaky Toy PCB

A custom single-layer PCB designed and fabricated using **KiCad** for a **555 Timer IC based touch-activated electronic toy**.

This project demonstrates the complete PCB design workflow, from schematic capture to PCB fabrication and hardware testing.

---

# 📖 Overview

The Queaky Toy is a simple electronic circuit that generates a sound when the touch input is activated. The PCB was designed in KiCad and fabricated using a CNC PCB milling machine.

This project was created to gain hands-on experience in PCB design, fabrication, soldering, and electronic hardware development.

---

# ✨ Features

* Single Layer PCB Design
* Designed using KiCad
* 555 Timer IC Based Circuit
* Touch Activated Input
* Buzzer Output
* CNC PCB Fabrication
* Through-Hole Components

# ⚙ Working Principle

The touch sensor provides a trigger signal to the 555 Timer IC. Once triggered, the timer generates an output signal that drives the buzzer, producing a sound. The circuit is powered using a 9V battery.
